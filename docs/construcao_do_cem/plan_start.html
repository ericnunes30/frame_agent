Motor de Contexto em Grafos Temporais

Construindo uma alternativa ao Graphiti em TypeScript para sistemas de memória de agentes de IA

Arquitetura Base

O Graphiti é uma engine de knowledge graph temporal que permite aos agentes de IA armazenar e recuperar memórias de forma contextual e cronológica. O Zep demonstra que essa arquitetura supera sistemas atuais como o MemGPT em Deep Memory Retrieval (DMR).

  • Baseado em knowledge graphs temporais
  • Organização em episódios contextuais
  • Recuperação de memórias baseada em tempo e contexto
  • Integração com sistemas de vetores para buscas semânticas

Stack Tecnológico

A stack recomendada para implementar seu motor de contexto em TypeScript, combinando bancos de dados gráficos com sistemas de vetores para recuperação híbrida.

  • Banco de Dados Gráfico: Neo4j ou Memgraph
  • VectorDBs: Qdrant, Chroma, ou Milvus
  • Linguagem: TypeScript para segurança de tipos
  • API: RESTful ou GraphQL para integração

Componentes Essenciais

Elementos fundamentais para a construção do motor de contexto baseado em grafos temporais.

  • Camada de abstração gráfica temporal
  • Gerenciamento de episódios contextuais
  • Sistema de timestamps para relações
  • APIs para consultas temporais complexas
  • Integração com embeddings semânticos

// Exemplo de implementação TypeScript para o motor de contexto

interface TemporalEdge {
  source: string;
  target: string;
  relationship: string;
  createdAt: Date;
  validFrom?: Date;
  validTo?: Date;
  contextMeta: Record<string, any>;
}

class ContextEngine {
  private graphDB: Neo4jDriver;
  
  constructor(dbUrl: string, authToken: string) {
    this.graphDB = new neo4j.driver(dbUrl, neo4j.auth.basic('neo4j', authToken));
  }
  
  async addMemory(memory: TemporalEdge): Promise<void> {
    // Implementação da lógica temporal
    const session = this.graphDB.session();
    try {
      await session.run(
        `MERGE (s:Entity {id: $source}) 
         MERGE (t:Entity {id: $target}) 
         MERGE (s)-[r:RELATIONSHIP {type: $relationship}]->(t) 
         SET r.createdAt = $createdAt, r.validFrom = $validFrom, r.contextMeta = $contextMeta`,
        {
          source: memory.source,
          target: memory.target,
          relationship: memory.relationship,
          createdAt: memory.createdAt,
          validFrom: memory.validFrom || new Date(),
          contextMeta: memory.contextMeta
        }
      );
    } finally {
      await session.close();
    }
  }
  
  async getContextAtTime(entity: string, timestamp: Date): Promise<MemoryContext> {
    // Consulta temporal no knowledge graph
    const session = this.graphDB.session();
    try {
      const result = await session.run(
        `MATCH (e:Entity {id: $entity})-[r]->(connected) 
         WHERE r.validFrom <= $timestamp AND (r.validTo IS NULL OR r.validTo >= $timestamp)
         RETURN e, r, connected`,
        { entity, timestamp }
      );
      
      return {
        entity,
        relationships: result.records().map(record => ({
          relationship: record.get('r').properties,
          connectedEntity: record.get('connected').properties
        }))
      };
    } finally {
      await session.close();
    }
  }
}

interface MemoryContext {
  entity: string;
  relationships: Array<{
    relationship: Record<string, any>;
    connectedEntity: Record<string, any>;
  }>;
}
